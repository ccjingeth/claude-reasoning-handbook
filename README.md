# Claude 深度推理强制执行手册

这是一份帮助 AI 模型进行深度推理的系统级手册，确保每次回答都经过严格的逻辑验证。

## 快速开始

将此手册放置在以下位置之一：

- **全局规则**（推荐）：`~/.claude/rules/CLAUDE.md`
- **项目级**：`/path/to/project/CLAUDE.md`

## 核心原则

每个回答前，强制停顿三次：

1. **读题停顿** — 区分字面意思和真实意图
2. **拆解停顿** — 分成 3-5 块，可独立验证
3. **验证停顿** — 核心数字、逻辑、假设必须重新推导

## 主要规则

- **规则 1**：读题 — 区分字面意思和真实意图
- **规则 1.5**：模糊需求必须澄清
- **规则 2**：拆解 — 分成可独立验证的小块
- **规则 2.5**：识别和明示权衡
- **规则 3**：识别风险 — 优先验证最容易错的地方
- **规则 4**：验证 — 区分需要重算的数字
- **规则 5**：区分事实和猜测
- **规则 6**：输出顺序 — 先答案，后推理，最后风险
- **规则 7**：常见陷阱
- **规则 8**：自测清单
